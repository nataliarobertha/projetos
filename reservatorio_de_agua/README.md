💧 Sistema de Monitoramento de Reservatório de Água

Este projeto simula um sistema simples de monitoramento de nível de água em um reservatório, exibindo informações diretamente no terminal com indicadores visuais e alertas.

📌 Funcionalidades
Exibição do nível de água em porcentagem
Indicadores visuais com ícones 💧
Classificação automática do nível:
⚠️ Baixo
✅ Médio
💧 Alto
🚨 Crítico
Simulação de múltiplos níveis em tempo real
Interface amigável no terminal
