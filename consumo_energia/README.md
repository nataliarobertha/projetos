# ⚡ Calculadora de Consumo de Energia

## 📌 Objetivo

Este programa calcula quanto um aparelho gasta de energia por mês.

## 💻 Linguagem

Python 🐍

## 🧮 Fórmula

(consumo = potência × horas × 30) ÷ 1000

## ▶️ Como usar

Digite no terminal:

python app.py

## ⚡ Exemplo

Geladeira → 150W → 10h/dia
