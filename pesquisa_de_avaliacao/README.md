# 📊 Pesquisa de Satisfação - empresa TudoWeb

Este é um projeto em **Python** desenvolvido para automatizar a coleta de dados de uma pesquisa de opinião para a empresa de marketing _TudoWeb_.

## 🚀 O que o programa faz?

O sistema organiza o atendimento de 50 clientes de forma rápida, coletando:

- Nome e Idade.
- Opinião sobre o atendimento (Excelente, Bom ou Ruim).

No final, o programa calcula e exibe automaticamente o total de respostas **Excelente** e o total de respostas **Ruim**.

## 🛠️ Tecnologias Utilizadas

- **Python 3**: Linguagem principal.
- **Estrutura de Repetição (`for`)**: Para repetir a pesquisa 50 vezes.
- **Estrutura de Decisão (`if/elif`)**: Para classificar as opiniões dos clientes.
